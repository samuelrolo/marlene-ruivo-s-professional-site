# Como Fazer um Workflow Correr

Este ficheiro é apenas um teste para demonstrar como workflows são acionados automaticamente.

## Quando um Workflow Executa?

Sempre que faz `git push origin development`, o workflow:

1. ✅ Detecta o push
2. ✅ Inicia automaticamente
3. ✅ Executa build
4. ⏸️ Aguarda aprovação (se configurado)
5. ✅ Faz deploy após aprovação
